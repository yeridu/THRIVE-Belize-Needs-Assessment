# Data

The deidentified student-level dataset (`tcc_data_paper1.csv`) and its codebook are **not** distributed in this repository, to protect the privacy of the participating students.

## How to request access

Contact the corresponding author (Mario Morales, mariomorales@arizona.edu). Access is subject to:

1. Approval by Toledo Community College and the local health partners.
2. A data-use agreement protecting student privacy.

## Placement

Once access is granted, place `tcc_data_paper1.csv` one directory **above** the repository root; the analysis script reads it as `../tcc_data_paper1.csv`. Alternatively, edit the `read.csv()` path at the top of `tcc_analysis4_paper1.Rmd`.
